## 三、公式
### 1、长公式及对齐
```
\begin{multline*}
 x=a+b+c+{} \\
   d+e+f+g
\end{multline*}
```

$$\begin{multline*}
x=a+b+c+{}\\
    d+e+f+g
\end{multline*}$$

```
\begin{split}
 x={} &a+b+c+{}\\
      &d+e+f+g
 \end{split}
 ```

$$\begin{split}
x={} &a+b+c+{}\\
       &d+e+f+g
  \end{split}$$
	   
### 2、多行公式及其对齐
```
\begin{gather}
  a=b+c+d\\
  z=x+y
\end{gather}
```

$$\begin{gather}
a=b+c+d\\
z=x+y
\end{gather}$$

```
\begin{align}
 a&=b+c+d\\
 z&=x+y
\end{align}
```

$$\begin{align}
a&=b+c+d\\
z&=x+y
\end{align}$$

### 3、分支公式
```
y=\begin{cases}
 -x,\quad x\leq 0\\
  x,\quad x>0
\end{cases}
```

$$y=\begin{cases}
       -x,\quad x\leq 0\\
	x,\quad x>0
	\end{cases}$$
